Generator to find the square numbers


    :param name: None
    :returns:  int -- the next square number in the sequence.
    :raises: AttributeError, KeyError

    :Example:

    This is called by using the wrapper function squares() to the
    ExpandingSequence class

    >>> squares = squares()
    >>> squares[5]
    >>> squares.seq()
    >>> len(squares)
    16
    [0, 1, 4, 9, 16]
    5
